# Icon

`MdIcon` renders a Material Symbols icon by name.

<div class="md-doc-preview">
  <MdIcon>star</MdIcon>
  <MdIcon>favorite</MdIcon>
  <MdIcon>settings</MdIcon>
</div>

## Import

```js
import { MdIcon } from 'vue-material-3';
```

## Usage

```vue
<MdIcon>star</MdIcon>
```
