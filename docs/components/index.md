# Components

Vue Material 3 currently includes these component groups:

| Group | Components |
| --- | --- |
| Autocomplete | `MdFilledAutoComplete`, `MdOutlinedAutoComplete` |
| Avatar | `MdAvatar` |
| Badge | `MdBadge` |
| Button | `MdElevatedButton`, `MdFilledButton`, `MdFilledTonalButton`, `MdOutlinedButton`, `MdTextButton` |
| Card | `MdCard` |
| Checkbox | `MdCheckbox` |
| Chips | `MdAssistChip`, `MdChipSet`, `MdFilterChip`, `MdInputChip`, `MdSuggestionChip` |
| Date Picker | `MdDatePicker`, `MdFilledDatePickerField`, `MdOutlinedDatePickerField` |
| Dialog | `MdDialog` |
| Divider | `MdDivider` |
| FAB | `MdBrandedFab`, `MdFab`, `MdFabExtended` |
| Icon | `MdIcon`, `MdIconButton` |
| List | `MdList`, `MdListDivider`, `MdListItem` |
| Menu | `MdMenu`, `MdMenuItem`, `MdSubMenu` |
| Navigation | `MdNavigationBar`, `MdNavigationDrawer`, `MdNavigationTab` |
| Progress | `MdCircularProgressIndicator`, `MdLinearProgressIndicator` |
| Selection | `MdRadio`, `MdSwitch` |
| Segmented Button | `MdSegmentedButton`, `MdSegmentedButtonSet` |
| Select | `MdFilledSelect`, `MdOutlinedSelect`, `MdSelectOption` |
| Slider | `MdSlider` |
| Snackbar | `MdSnackbar` |
| Tabs | `MdPrimaryTab`, `MdSecondaryTab`, `MdTabs` |
| Text Field | `MdFilledTextField`, `MdOutlinedTextField` |
| Time Picker | `MdTimePicker`, `MdFilledTimePickerField`, `MdOutlinedTimePickerField` |
| Tooltip | `MdTooltip` |

Interactive examples should live in Storybook. This page can grow into per-component usage and API documentation.
