# Badge

Badges show small counts or status markers on top of another UI element.

  </span>
  <span class="md-doc-preview__badge-target">

    <MdBadge absolute />
  </span>
</div>

## Import

```js
import { MdBadge } from 'vue-material-3';
```

## Usage

```vue
<span style="position: relative">
  <MdIcon>mail</MdIcon>
  <MdBadge value="10" absolute />
</span>
```
